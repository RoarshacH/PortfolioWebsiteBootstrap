About

A password is a string of human-readable characters that are used mainly for authentication in computer systems. Currently, most of the websites use passwords as their main authentication mechanism. Many past studies have shown that passwords can easily be hacked, bypassed with some effort from a third party (Zviran and Haga 1999).

Rather than using a string of letters that the user has to remember, using the user’s fingerprint or their mobile device as the main authentication mechanism has the possibility in reducing the inconvenience the user faces when creating and remembering secure passwords and increase the security of the website and the user’s data. (Bonneau et al. 2012).

In this research project, I try to answer the question is it feasible to implement an alternative authentication mechanism in a website. If so, how will the users react to that system? This project aims to measure the performance and viability of such a system by implementing it on a website with the created authentication system and running a controlled trial using selected participants.
